# VTL Scripts for API Gateway

This folder contains all tools and documentation for managing VTL (Velocity Template Language) scripts in API Gateway.

## 📁 Files

- **`vtl-helper.sh`** - Helper script for common VTL operations
- **`README-VTL.md`** - Complete guide with all commands and examples
- **`QUICK_START.md`** - Quick reference guide
- **`vtl-setup-guide.md`** - Detailed setup guide
- **`sample-request-template.vtl`** - Example request transformation template
- **`sample-response-template.vtl`** - Example response transformation template

## 🚀 Quick Start

```bash
cd /home/unknown/projects/calytics/scripts/vtl
./vtl-helper.sh list-apis
```

See `README-VTL.md` for complete documentation.
